# Typing-Speed-Test

## Description:
 This is a web based typing speed test inspired from typing.com. It tracks typing performance in real-time, displaying my current WPM, accuracy and total keystroes. 

## Why is built this:
i built this project to act as a personal tool for improving my typing speed through consistent and focuses practice rather than displaying ads everywhere which i never liked on other websites.

## Tech Stack:
 * HTML: Core structure for the layout.
 * CSS: custom styling css variables, specific design choices to emulate a clean, retro aesthetic web.
 * Javascript: core logic, including word randomization, timer management,live stat calculation,etc.

## How it works:
 This generates a randomized stream of words within a scrollable display box. As you type into the input field, the script monitors your keystrokes, calculates your WPM and accuracy in real-time, and automatically scrolls the text to keep your current word in view. upon reaching the time Limit, a window appears, presenting yout final stats.
